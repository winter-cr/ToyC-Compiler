## 四、剩余问题分析

### f16_complex_syntax（编译器异常）

无 OJ 测试用例，推测可能原因：
- 深度嵌套的表达式或语句导致 Parser 递归栈溢出
- 某种语法组合触发了 AstToIr 未处理的节点类型
- 语义分析对复杂结构的检查有遗漏导致后续代码生成失败

### f18_many_variables（编译器异常）

无 OJ 测试用例，推测可能原因：
- 大量局部变量使栈帧超过 `fitsImmediate12` 范围
- Slot 分配冲突或偏移计算溢出
- 变量数超过 `createLocal` 的 uint32_t 上限（但几乎不可能）
