## Additional Optimization Notes

- Tail recursion is optimized into an intra-function loop for small self-recursive calls followed immediately by return.
- Compare-and-branch fusion reduces loop condition overhead.
- Temporary-to-local copy coalescing lets common loop updates write directly to local registers.
